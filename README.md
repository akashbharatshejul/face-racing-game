# AI Face Controlled Car Racing Game

A Python car racing game controlled using face gestures.

## Features

- Head movement to steer car
- Blink detection for shield
- Mouth open to slow time
- Emotion detection (smile / surprise)
- Dynamic weather system
- Face-controlled menu
- High score saving
- Sound effects and animations

## Tech Stack

Python  
OpenCV  
MediaPipe  
Pygame  

## Controls

Head Left / Right → Move car  
Blink → Activate shield  
Open Mouth → Slow motion  
Smile → Snow weather  
Surprise → Sunny weather
